A rudimentary todo list manager written from scratch in Python.
Accepts the following commands:

  add   - Adds an entry to the list.
  rem   - Removes an entry from the list by index number.
  list  - Prints the list of entries currently in the list, with
          the index number and description of each one.
  sort  - Sorts the entries alphabetically.
  exit  - Exits the program.
